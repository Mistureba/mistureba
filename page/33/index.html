League of Legends no Ubuntu

Este guia finalmente foi atualizado – ou quase isso…

Na verdade, você não vai ver nem metade das coisas que aconteceram no trailer dentro do jogo em si, mas ele é fodástico mesmo assim :p

Configurações Iniciais

Antes de partir para a instalação do League of Legends, será preciso fazer algumas configurações iniciais:

a) Certifique-se de estar usando o Driver mais atualizado de sua placa de vídeo: apesar da maioria das distribuições Linux vir com esses drivers por padrão, eles geralmente não estão ativados nem atualizados. Com o driver padrão, o LoL fica bugado. Antes de fazer a instalação você vai precisar instalar um pacote primeiro:

$ sudo apt-get update && sudo apt-get install linux-headers-generic

Agora, para fazer a instalação dos drivers, use a seguinte linha de comando:

$ sudo add-apt-repository ppa:ubuntu-x-swat/x-updates -y && sudo apt-get update && sudo apt-get install nvidia-current

Então reinicie o computador. Se tudo der certo, você poderá alterar a resolução de sua tela para o máximo permitido por seu monitor sem que a imagem fique granulada.

Porém, pode ser que as letras fiquem miúdas demais para ver, mesmo sem mudar a resolução. Nesse caso, use a seguinte linha de comando:

$ sudo nvidia-xconfig --no-use-edid-dpi

E reinicie. As letras ficarão legíveis novamente.

b) Certifique-se de estar usando as dependencias mais atuais: o Wine é um programa que fornece algo chamado “camada de compatibilidade” a programas Windows rodando no Ubuntu, enquanto o Mono é uma alternativa open source ao .NET da Microsoft. Já cURL, mesa-utils e libtxc-dxtn0 são programas que o Playonlinux usa para baixar e testar instaladores e patches. Vamos precisar deles para rodar o League of Legends. O comando abaixo adiciona o repositório mais atual do Wine e instala todos os pacotes:

$ sudo add-apt-repository ppa:ubuntu-wine/ppa -y && sudo apt-get update && sudo apt-get install wine mono-complete curl mesa-utils libtxc-dxtn0

É importante instalar o Wine apartir do repositório acima porque o Wine disponível no repositório do Ubuntu não consegue lidar corretamente com a versão mais atual do Mono.

c) Certifique-se de estar utilizando a fonte mais atualizada do Playonlinux: algo que faz do LoL um jogo fantástico acaba por torná-lo uma dor de cabeça aos administradores do Playonlinux, que é sua atualização constante. Para evitar dores de cabeça – deixando-as pra equipe de suporte do Playonlinux – sugiro adicionar a fonte do Playonlinux ao seu Ubuntu, assim ele atualiza automaticamente o programa quando puder.

Primeiro, desinstale o Playonlinux caso o tenha instalado na sua máquina:

$ sudo apt-get remove playonlinux

A linha de comando abaixo serve para adicionar a fonte oficial do Playonlinux:

Tenha o cuidado de escolher a linha referente à sua distribuição!

Ubuntu 10.04 (Lucid Lynx)

$ sudo wget https://deb.playonlinux.com/playonlinux_lucid.list -O /etc/apt/sources.list.d/playonlinux.list

Ubuntu 10.10 (Maverick Meerkat)

$ sudo wget https://deb.playonlinux.com/playonlinux_maverick.list -O /etc/apt/sources.list.d/playonlinux.list

Ubuntu 11.04 (Natty Narwhal)

$ sudo wget https://deb.playonlinux.com/playonlinux_natty.list -O /etc/apt/sources.list.d/playonlinux.list

Ubuntu 11.10 (Oneiric Ocelot)

$ sudo wget https://deb.playonlinux.com/playonlinux_oneiric.list -O /etc/apt/sources.list.d/playonlinux.list

Ubuntu 12.04 (Precise Pangolin)

$ sudo wget https://deb.playonlinux.com/playonlinux_precise.list -O /etc/apt/sources.list.d/playonlinux.list

Ubuntu 12.10 (Quantal Quetzal)/ 13.04 (Raring Ringtail)

$ sudo wget https://deb.playonlinux.com/playonlinux_quantal.list -O /etc/apt/sources.list.d/playonlinux.list

Finalmente, esta linha irá instalar o Playonlinux em si:

$ sudo apt-key adv –keyserver keyserver.ubuntu.com –recv-keys E0F72778C4676186 -y && sudo apt-get update && sudo apt-get install playonlinux

Abra o PlayOnLinux e quando ele terminar de verificar por atualizações, pause para respirar. A interface inicial deve ser parecida com isso:

Só que sem nada instalado… ainda.

d) Baixe o instalador offline em Português Brasileiro: Apesar do Playonlinux ser capaz de baixar um instalador pra você, ele não possuirá a opção de mudar para o servidor Brasileiro. Há uma forma de se conseguir esse instalador no Ubuntu, mas como o foco deste tutorial é a instalação do League of Legends no Ubuntu, basta fazer o download do instalador pt-br aqui.

Instalando o League of Legends

Ok, agora a parte realmente chata. Primeiro, clique no botão com um sinal de “+”. Vai abrir a seguinte janela:

Então, marque a caixa escrita “testaveis” que está ao lado da caixa de busca:

Vai aparecer uma mensagem explicando que o suporte do Playonlinux ao League of Legends é experimental, e que eles não têm certeza se fazer isso irá ou não aumentar o efeito estufa. Clique em ok.

Escreva “league” na caixa de busca, isso deve fazer a entrada do script do League of Legends aparecer que nem na imagem abaixo. Então, clique em “instalar”.

Tudo o que você precisa fazer depois disso é seguir as instruções do instalador, tendo os seguintes cuidados:

  • Caso a idéia seja jogar no servidor BR, você vai precisar usar um instalador pt-br completo baixado anteriormente. Quando o instalador finalmente iniciar, siga as instruções mas, no final, DESMARQUE a opção de iniciar o League of Legends automaticamente após a instalação. Ela aparece na última tela do instalador, junto com um Teemo (seria coincidência?):

  • Se resolver deixar o Playonlinux baixar um instalador pra você, você precisará instalar o pacote p7zip-full, ou o Playonlinux não vai conseguir descompactar o instalador (sim, isso aconteceu comigo. Shit happens). Além disso, o instalador será descompactado na sua Área de Trabalho e estará no idioma en-us (portanto, sem opção pelo server BR). Guarde esse instalador com carinho, pois evita que você tenha que baixá-lo novamente;

Por fim, o Playonlinux vai criar o atalho para iniciar o programa e então teremos que fazer os ajustes finais.

Ajustes Finais

Você deve estar seco pra jogar, mas sugiro fazer os seguintes ajustes antes de começar:

  • Na interface do Playonlinux, clique sobre o lançador do League of Legends e vá na opção “Configurar/Wine/Configurar Wine”. Uma nova janela vai abrir, nela você deve clicar na aba “Graphics” e ativar a opção “emulate a virtual desktop”, usando a resolução atual de seu monitor como dimensões. Veja abaixo:

  • Abra o cliente do League of Legends e, se tiver usado o instalador pt-br, mude o servidor para “Brazil” e idioma para pt-br. Espere o cliente atualizar e então jogue. Só para lembrar, se você tiver utilizado o instalador baixado pelo Playonlinux, você provavelmente não poderá fazer isso;

  • Quando terminar a atualização, você possui a opção de instalar um patch chamado “ingame shopping icons fix”. Sem isso, algumas imagens de ítens são substituidas por caixas vermelhas durante as partidas. O patch é pequeno, mas demora alguns minutos para ser aplicado, então seja paciente. Além disso, o patch precisa ser reinstalado a cada atualização de temporada.

Bem, o tutorial acabou. Quaisquer dúvidas serão bem vindas nos comentários. Deixo pra vocês a coisa mais nonsense que já vi numa partida de League of Legends:

Battlestar Galactica LARP

Serei franco com vocês: os “LARP’s” causaram mais dano à imagem do RPG do que os Criacionistas causaram ao Cristianismo. A primeira impressão de quem vêm à pesquisar sobre LARP é a de um bando de nerds perdedores, vestindo fantasias improvisadas pelas mães, correndo em um bosque gritando “Lightning bolt”.

Sério, não veja!

Claro que nem sempre é assim. Uma boa partida de LARP é um exercício de interpretação, uma chance de viver em uma realidade diferente por alguns minutos. É uma experiência fantasiosa tão gratificante quanto produzir um filme ou ler um livro – talvez até mais.

Esse tipo de diversão é mais comum na Europa. Há até mesmo casos de sessões de LARP’s que foram filmadas e compradas por emissoras de TV, para então serem transmitidas como seriados.

The Monitor Celestra é de longe o mais ambicioso LARP de que se têm notícia. Com um orçamento de mais de 1 milhão de coroas suecas (cerca de R$325.600,00) mais um destroyer naval aposentado, uma equipe de designers organizou um RPG Live Action – comumente conhecido como LARP – capaz de deixar muita série global no chinelo.

The Monitor Celestra começou como uma recriação do Battlestar Galactica original de Ronald D. Moore, mas os organizadores resolveram mover a ação para o Celestra, uma nave que foi citada mas quase nunca vista no próprio show. Durante alguns dias de março, LARPers interpretaram um conflito tenso e por vezes mortal entre a tripulação civil do Celestra e um grupo de abordagem militar, o tempo todo tentando desvendar os Cylons em seu meio. Um dos três finais de semana do jogo terminou com a rendição dos humanos aos agentes Cylon, outro atolou em uma amarga limpeza étnica, e o terceiro – que teve a menor quantidade de mortes – resultou na própria nave ficcional explodindo.

“Dizer a palavra LARP é uma ótima maneira de perder a atenção de alguém em 30 segundos”. Martin Ericsson.

Mas se depender dos seus criadores, o Gamemaster Martin Ericsson e a produtora Cecilia Dolk, o próximo destino do Celestra e sua tripulação será a América. Mesmo muitos anos após sair do ar, Battlestar Galactica ainda goza de uma grande base de fãs americanos. O elenco original, inclusive, elogiou a sua adaptação – Michael Hogan (Coronel Tigh) ainda manifestou interesse em participar da nova tentativa. Então o que está atrapalhando a empreitada? O fato de que é um LARP.

Na Suécia e outros países nórdicos, LARP é relativamente aceito como uma forma de jogo ou arte de improviso – algumas LARPs até são financiadas por doações do governo. Mas, nos EUA, ainda é uma piada (Brasil, nem se fala). Ericsson lamenta os vídeos do YouTube mostrando LARPers mal trajados gritando feitiços.

“Esse video do lightning bolt criou mais problemas para a comunidade LARP dos EUA do que qualquer outro vídeo”. Martin Ericsson.

Ericsson, Dolk, e um grupo de entusiastas americanos estão tentando mudar esse cenário. Para tanto, basta simplesmente aumentar a visibilidade do LARP na casa: ano passado, um grupo interpretou Mad About the Boy, possivelmente o primeiro LARP Europeu em solo americano. Além disso, será preciso preparar o público americano para reformular o debate.

“É apenas uma forma diferente de expressão. Ainda é um LARP. Ainda é um jogo”. Cecília Dolk.

Atualmente, não há data definida para uma turnê americana. O projeto será provisoriamente “lançado” de alguma forma em 2014, mas a equipe ainda não decidiu onde realizar o jogo ou como financiá-lo. Além de uma pequena bolsa financiada por Dolk, as sessões originais do The Monitor Celestra foram pagas pelos participantes. Desta vez, a equipe também considerou a tentativa de obter ajuda dos próprios donos do show. O jogo não foi oficialmente licenciado, mas Ericsson não acha que isso vai ser um problema – especialmente porque Moore e os outros estão claramente cientes do projeto.

“Qualquer um pode fazer cosplay de Starbuck, não há nenhum problema de direitos autorais nisso. Nós estamos na verdade fazendo a mesma coisa só que em grande escala”. Diz Ericsson.

Se The Monitor Celestra for executado novamente, ele vai expor ao público americano uma nova faceta do mundo de Battlestar Galactica, apimentado pelos recentes debates suecos sobre racismo e nacionalismo. “Se a principal alegoria de BSG é o pós-9/11, sobre liberdade, segurança e o trauma do 11/9 trouxe com ele, então nossa principal alegoria teria que ser sobre algo que seja relevante hoje na Europa”.

A equipe optou por se concentrar no conflito cultural entre os dois maiores grupos étnicos dentro do universo de BSG: os urbanos e individualistas Capricanos (nativos do planeta Caprica); e os Tauranos (nativos do planeta Tauron), mais rústicos e tradicionais. Mais detalhes sobre eles no seriado Caprica.

Ericsson espera explorar o desconforto que motivou grupos de extrema-direita a surgirem, como os democratas suecos. “O aumento dos partidos ultra-conservadores, francamente racistas, é o maior problema na Europa neste momento”, diz ele. “Encontramos duas culturas que achamos que dariam conta de representá-los sobre estas questões.”

The Monitor Celestra é em parte improvisação, com mecânicas projetadas para moldar a história em uma vasta gama de resultados possíveis. Ericsson descreve-o como uma série de caixas de areia cheias de diferentes histórias possíveis. Uma pessoa pode acabar em uma luta para apresentar um trabalho, enquanto outro poderia ter um relacionamento intra-jogo, enquanto trabalha em um laboratório secreto de Inteligência Artificial.

Mas por trás das interações sociais encontra-se um rigoroso sistema para simular a vida em uma nave espacial ficcional. Na partida original, 10 computadores em rede e 12 Raspberry Pis foram usados para controlar todas as áreas nas quais Celestra acontecia. A fim de manter a ação constante entre os jogadores, coisas como coordenadas e orientação tiveram que ser transmitida entre os diferentes departamentos fisicamente, mas os jogadores foram ajudados por consoles Arduino potentes e software personalizado apartir de um simulador de submarino chamado Silent Hunter.

Para Ericsson e Dolk, Battlestar Galactica é o cenário perfeito para um LARP. “É tanto drama de caráter íntimo e comentários sobre sociopolítica e batalhas espaciais impressionantes”, diz Ericsson. Um renascimento americano lhe permitiria trazer o LARP europeu para os EUA sem a pressão de representar todo um gênero como o steampunk ou larping pós-apocalíptico. Isso dará à equipe a chance de corrigir problemas que ocorreram com as primeiras partidas – principalmente o fato de que eles sentiram que muitos dos mais de 100 personagens foram interpretados bem o suficiente.

“Estamos apenas fazendo algo super-específico: um show em um ambiente. E depois veremos se as pessoas gostam.”

Já sinto o gosto de RPG Indie. Até a próxima!
