League of Legends no Ubuntu

Este guia finalmente foi atualizado – ou quase isso…

Na verdade, você não vai ver nem metade das coisas que aconteceram no trailer dentro do jogo em si, mas ele é fodástico mesmo assim :p

Configurações Iniciais

Antes de partir para a instalação do League of Legends, será preciso fazer algumas configurações iniciais:

a) Certifique-se de estar usando o Driver mais atualizado de sua placa de vídeo: apesar da maioria das distribuições Linux vir com esses drivers por padrão, eles geralmente não estão ativados nem atualizados. Com o driver padrão, o LoL fica bugado. Antes de fazer a instalação você vai precisar instalar um pacote primeiro:

$ sudo apt-get update && sudo apt-get install linux-headers-generic

Agora, para fazer a instalação dos drivers, use a seguinte linha de comando:

$ sudo add-apt-repository ppa:ubuntu-x-swat/x-updates -y && sudo apt-get update && sudo apt-get install nvidia-current

Então reinicie o computador. Se tudo der certo, você poderá alterar a resolução de sua tela para o máximo permitido por seu monitor sem que a imagem fique granulada.

Porém, pode ser que as letras fiquem miúdas demais para ver, mesmo sem mudar a resolução. Nesse caso, use a seguinte linha de comando:

$ sudo nvidia-xconfig --no-use-edid-dpi

E reinicie. As letras ficarão legíveis novamente.

b) Certifique-se de estar usando as dependencias mais atuais: o Wine é um programa que fornece algo chamado “camada de compatibilidade” a programas Windows rodando no Ubuntu, enquanto o Mono é uma alternativa open source ao .NET da Microsoft. Já cURL, mesa-utils e libtxc-dxtn0 são programas que o Playonlinux usa para baixar e testar instaladores e patches. Vamos precisar deles para rodar o League of Legends. O comando abaixo adiciona o repositório mais atual do Wine e instala todos os pacotes:

$ sudo add-apt-repository ppa:ubuntu-wine/ppa -y && sudo apt-get update && sudo apt-get install wine mono-complete curl mesa-utils libtxc-dxtn0

É importante instalar o Wine apartir do repositório acima porque o Wine disponível no repositório do Ubuntu não consegue lidar corretamente com a versão mais atual do Mono.

c) Certifique-se de estar utilizando a fonte mais atualizada do Playonlinux: algo que faz do LoL um jogo fantástico acaba por torná-lo uma dor de cabeça aos administradores do Playonlinux, que é sua atualização constante. Para evitar dores de cabeça – deixando-as pra equipe de suporte do Playonlinux – sugiro adicionar a fonte do Playonlinux ao seu Ubuntu, assim ele atualiza automaticamente o programa quando puder.

Primeiro, desinstale o Playonlinux caso o tenha instalado na sua máquina:

$ sudo apt-get remove playonlinux

A linha de comando abaixo serve para adicionar a fonte oficial do Playonlinux:

Tenha o cuidado de escolher a linha referente à sua distribuição!

Ubuntu 10.04 (Lucid Lynx)

$ sudo wget https://deb.playonlinux.com/playonlinux_lucid.list -O /etc/apt/sources.list.d/playonlinux.list

Ubuntu 10.10 (Maverick Meerkat)

$ sudo wget https://deb.playonlinux.com/playonlinux_maverick.list -O /etc/apt/sources.list.d/playonlinux.list

Ubuntu 11.04 (Natty Narwhal)

$ sudo wget https://deb.playonlinux.com/playonlinux_natty.list -O /etc/apt/sources.list.d/playonlinux.list

Ubuntu 11.10 (Oneiric Ocelot)

$ sudo wget https://deb.playonlinux.com/playonlinux_oneiric.list -O /etc/apt/sources.list.d/playonlinux.list

Ubuntu 12.04 (Precise Pangolin)

$ sudo wget https://deb.playonlinux.com/playonlinux_precise.list -O /etc/apt/sources.list.d/playonlinux.list

Ubuntu 12.10 (Quantal Quetzal)/ 13.04 (Raring Ringtail)

$ sudo wget https://deb.playonlinux.com/playonlinux_quantal.list -O /etc/apt/sources.list.d/playonlinux.list

Finalmente, esta linha irá instalar o Playonlinux em si:

$ sudo apt-key adv –keyserver keyserver.ubuntu.com –recv-keys E0F72778C4676186 -y && sudo apt-get update && sudo apt-get install playonlinux

Abra o PlayOnLinux e quando ele terminar de verificar por atualizações, pause para respirar. A interface inicial deve ser parecida com isso:

Só que sem nada instalado… ainda.

d) Baixe o instalador offline em Português Brasileiro: Apesar do Playonlinux ser capaz de baixar um instalador pra você, ele não possuirá a opção de mudar para o servidor Brasileiro. Há uma forma de se conseguir esse instalador no Ubuntu, mas como o foco deste tutorial é a instalação do League of Legends no Ubuntu, basta fazer o download do instalador pt-br aqui.

Instalando o League of Legends

Ok, agora a parte realmente chata. Primeiro, clique no botão com um sinal de “+”. Vai abrir a seguinte janela:

Então, marque a caixa escrita “testaveis” que está ao lado da caixa de busca:

Vai aparecer uma mensagem explicando que o suporte do Playonlinux ao League of Legends é experimental, e que eles não têm certeza se fazer isso irá ou não aumentar o efeito estufa. Clique em ok.

Escreva “league” na caixa de busca, isso deve fazer a entrada do script do League of Legends aparecer que nem na imagem abaixo. Então, clique em “instalar”.

Tudo o que você precisa fazer depois disso é seguir as instruções do instalador, tendo os seguintes cuidados:

  • Caso a idéia seja jogar no servidor BR, você vai precisar usar um instalador pt-br completo baixado anteriormente. Quando o instalador finalmente iniciar, siga as instruções mas, no final, DESMARQUE a opção de iniciar o League of Legends automaticamente após a instalação. Ela aparece na última tela do instalador, junto com um Teemo (seria coincidência?):

  • Se resolver deixar o Playonlinux baixar um instalador pra você, você precisará instalar o pacote p7zip-full, ou o Playonlinux não vai conseguir descompactar o instalador (sim, isso aconteceu comigo. Shit happens). Além disso, o instalador será descompactado na sua Área de Trabalho e estará no idioma en-us (portanto, sem opção pelo server BR). Guarde esse instalador com carinho, pois evita que você tenha que baixá-lo novamente;

Por fim, o Playonlinux vai criar o atalho para iniciar o programa e então teremos que fazer os ajustes finais.

Ajustes Finais

Você deve estar seco pra jogar, mas sugiro fazer os seguintes ajustes antes de começar:

  • Na interface do Playonlinux, clique sobre o lançador do League of Legends e vá na opção “Configurar/Wine/Configurar Wine”. Uma nova janela vai abrir, nela você deve clicar na aba “Graphics” e ativar a opção “emulate a virtual desktop”, usando a resolução atual de seu monitor como dimensões. Veja abaixo:

  • Abra o cliente do League of Legends e, se tiver usado o instalador pt-br, mude o servidor para “Brazil” e idioma para pt-br. Espere o cliente atualizar e então jogue. Só para lembrar, se você tiver utilizado o instalador baixado pelo Playonlinux, você provavelmente não poderá fazer isso;

  • Quando terminar a atualização, você possui a opção de instalar um patch chamado “ingame shopping icons fix”. Sem isso, algumas imagens de ítens são substituidas por caixas vermelhas durante as partidas. O patch é pequeno, mas demora alguns minutos para ser aplicado, então seja paciente. Além disso, o patch precisa ser reinstalado a cada atualização de temporada.
