Mx Masters – WordPress volume 2 – Desenvolvendo Portais (em mp4 DVD Rip)

Interessante Curso de WordPress em dois volumes (cujo primeiro volume é praticamente impossível de achar de graça pela net, sorry). Achei ele muito bom, apesar de não ser tão completo quanto o curso de WordPress da UpInside.

Ficha Técnica

Nome do Curso: WordPress volume 2 – Desenvolvendo Portais

Nome do Instrutor: Felipe Cardozo

Ano de Lançamento: 2009

Editora: Mx Cursos

Tamanho: 404 mb

Formato: mp4

Idioma: Português

Adicionado Arquivo SFV, que permite testar os arquivos baixados para checar por partes corrompidas. Deixem as partes baixadas e o arquivo SFV na mesma pasta, então abram o arquivo SFV com o programa RapidCRC ou semelhante.

Clique na imagem acima para ser redirecionado à pasta de download do Mediafire. Ei reencodei todos os vídeos para facilitar ainda mais o download. Sugiro que usem o K-Lite Mega Codec Pack para ver esses vídeos.

Se quiser, use um dos Mirrors abaixo:

Peeje.Com

Faz algum tempo que este Mirror vêm usando “slots” de download. Caso apareça a mensagem “All download slots for this file are currently filled. Please try again momentarily“, tenha paciência ou baixe de outro Mirror.

Links apagados, pois este mirror passou a adotar uma estratégia idiota: converter os link’s automaticamente de públicos para privados sem a autorização dos usuários, impedindo os downloads.

Jumbofiles.com

Este mirror apagou todos os arquivos que eu havia subido nele. Apagado.
