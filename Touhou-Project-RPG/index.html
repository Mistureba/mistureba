Touhou Project RPG

No dia 12 deste mês, Sascha Borges Lucas liberou a primeira edição de sua adaptação não-oficial de Touhou para 3D&T Alpha. Este e-book de 74 páginas possui informações suficientes para se jogar uma campanha em Gensokyo. Link para download:

Touhou RPG by Sascha

O destaque fica para a lista de NPC’s prontos, que inclui a história e descrição das principais personagens vistas nos jogos, e para a enorme pesquisa feita sobre o cenário.

Parabéns ao Sascha por ter realizado este trabalho e desejo que ele faça bastante sucesso entre os fãs de Touhou.
