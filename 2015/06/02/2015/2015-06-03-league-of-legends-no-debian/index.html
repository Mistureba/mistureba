League of Legends no Debian

Após um bom tempo sem postar nada, trago minha experiencias com o LoL no Linux. Porém, este novo tutorial não ensina como instalar os drivers de video. Há tantas possibilidades que me deu preguiça de mastigar essa informação.

Em compensação, há uma nota de como me livrei dos problemas de áudio no wine no final do tutorial.

Configurações Iniciais

Antes de partir para a instalação do League of Legends, será preciso fazer algumas configurações iniciais:

a) Baixe o instalador offline em Português Brasileiro: Apesar do Playonlinux ser capaz de baixar um instalador pra você, ele não possuirá a opção de mudar para o servidor Brasileiro. Há uma forma de se conseguir esse instalador por vias excusas, mas como o foco deste tutorial é a instalação do League of Legends, basta fazer o download do instalador pt-br aqui.

b) Certifique-se de estar usando as dependencias mais atuais: o Wine é um programa que fornece algo chamado “camada de compatibilidade” a programas Windows rodando no Ubuntu, enquanto o Mono é uma alternativa open source ao .NET da Microsoft. O Pulseaudio é um servidor de som local necessário para fazer o som do LoL funcionar. Já cURL, git, mesa-utils, python e libtxc-dxtn0 são programas que o Playonlinux usa para rodar, baixar e testar instaladores e patches. Vamos precisar deles para rodar o League of Legends. O comando abaixo instala todos os pacotes:


$ sudo dpkg –add-architecture i386 -y && sudo apt-get update && sudo apt-get install mono-complete curl git libtxc-dxtn0 mesa-utils python pulseaudio wine:i386

É importante instalar o Wine para a arquitetura i386 (32bits) mesmo que esteja usando um processador de última geração (64 bits), pois o Wine para outras arquiteturas simplesmente não funciona junto com o Playonlinux.

c) Download do Executável: a idéia aqui é adquirir o Playonlinux mais atualizado possível, então que tal baixar a versão de desenvolvimento?


$ cd && git clone https://github.com/PlayOnLinux/POL-POM-4

O comando acima baixa uma pasta chamada “POL-POM-4” direto para a sua pasta Home. Então, basta dar um clique duplo no arquivo “playonlinux” que a interface do programa se abrirá:

Instalando o League of Legends

Ok, agora a parte realmente chata. Primeiro, clique no botão com um sinal de “+” escrito “Instalar”. Vai abrir a seguinte janela:

Escreva “league” na caixa de busca, isso deve fazer a entrada do script do League of Legends aparecer que nem na imagem abaixo. Então, clique em “instalar”.

Vai aparecer uma mensagem explicando que o suporte do Playonlinux ao League of Legends é experimental, e que eles não têm certeza se o instalador irá ou não explodir seu computador.

Clique em ok e siga as instruções do instalador, tendo os seguintes cuidados:

  • Caso a idéia seja jogar no servidor BR, você vai precisar usar um instalador pt-br completo baixado anteriormente. Quando o instalador finalmente iniciar, siga as instruções mas, no final, DESMARQUE a opção de iniciar o League of Legends automaticamente após a instalação. Ela aparece na última tela do instalador, junto com um Teemo (seria coincidência?):

  • Por algum motivo, o Playonlinux não baixa um instalador compatível com o servidor brasileiro quando você pede para fazer o download. Melhor deixar quieto;

Por fim, o Playonlinux vai criar o atalho para iniciar o programa e então teremos que fazer os ajustes finais.

Ajustes Finais

Você deve estar seco pra jogar, mas sugiro fazer os seguintes ajustes antes de começar:

  • Na interface do Playonlinux, clique sobre o lançador do League of Legends e vá na opção “Configurar/Wine/Configurar Wine”. Uma nova janela vai abrir, nela você deve clicar na aba “Gráficos” e ativar a opção “emular uma área de trabalho virtual”, usando a resolução atual de seu monitor como dimensões. Veja abaixo:

  • Abra o cliente do League of Legends e, se tiver usado o instalador pt-br, mude o servidor para “Brazil” e idioma para pt-br. Espere o cliente atualizar e então jogue. Só para lembrar, se você tiver utilizado o instalador baixado pelo Playonlinux, você provavelmente não poderá fazer isso;

  • Caso o som da tela de login – e de todo o resto da área virtual – esteja acelerada, você precisará adicionar as seguintes linhas ao arquivo “/etc/pulse/daemon.conf”:


    default-fragments: 5
    default-fragment-size-msec: 2
    realtime-scheduling=true

    Então use o comando abaixo para reiniciar o pulseaudio:


    $ pulseaudio -k && pulseaudio –start

    Por fim, crie um arquivo chamado “.asoundrc” na sua pasta Home (precisa ser exatamente esse nome e posição) com o seguinte conteúdo:


    pcm.pulse {
    type pulse
    }
    ctl.pulse {
    type pulse
    }
    pcm.!default {
    type pulse
    }
    ctl.!default {
    type pulse
    }

Você provavelmente se verá livre dos problemas de audio do Wine.
